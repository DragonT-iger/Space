# 🎒공간 대여 플랫폼 SPACE

Java와 Spring MVC를 기반으로 제작한 백엔드 웹 개발자 과정 최종 프로젝트입니다.

일반유저, 등록유저, 관리자 파트로 기능을 나누어 플랫폼 형태의 서비스를 구축했습니다.
